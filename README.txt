Для запуска приложения:
1) перейти в подпапку "target" в корневой папке проекта
2) скопировать содержимое файла с тестовыми сущностями в файл input.txt, находящийся в этой папке
3) запустить консоль
4) перейти в консоли в подпапку "target" в корневой папке проекта
5) выполнить команду "java -jar TestTask-1.0-SNAPSHOT.jar"
Результаты выполнения программы будут в файле с названием results_<дата и время>.txt в этой же подпапке.
